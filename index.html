Cascading Style Sheets atau CSS adalah bahasa pemograman berbasis aturan yang di kembangkan oleh World Wide Web Consortium (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi untuk memformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut.

Misalnya, pada halaman HTML versi 3.2, sudah di perkenalkan tag agar gaya teks bisa di ubah. Namun, web designer amatir menyalahgunakannya dan tidak ada opsi fallback seperti yang sekarang tersedia di CSS. Jadi, pilihan terbaiknya adalah tetap memisahkan pemformatan (HTML) dan penataan gaya (CSS)
